anomalib.models.get_model(config: Union[omegaconf.DictConfig, omegaconf.ListConfig]) → anomalib.core.model.AnomalyModule[source]¶

Load model from the configuration file.

Works only when the convention for model naming is followed.

The convention for writing model classes is anomalib.models.<model_name>.model.<Model_name>Lightning anomalib.models.stfpm.model.StfpmLightning

and for OpenVINO anomalib.models.<model-name>.model.<Model_name>OpenVINO anomalib.models.stfpm.model.StfpmOpenVINO

Parameters
